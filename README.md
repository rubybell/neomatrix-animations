# neomatrix-animations
A collection of animations, vibe-coded with Claude, for the [Adafruit NeoMatrix 8x8](https://www.adafruit.com/product/1487), written in MakeCode JavaScript for micro:bit V2.

### 🎵 Radial Audio Visualiser
A program that uses the loudness from the microphone on the micro:bit to scale the radius of a rainbow circle. This one is nice to listen to [with sound.](https://vimeo.com/1174860756/a7080c758c?share=copy&fl=sv&fe=ci)
